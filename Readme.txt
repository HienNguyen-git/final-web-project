Hướng dẫn các chạy chương trình:
	* Điều kiện tiên quyết:
		- Có cài đặt nodeJS
		- Cài đặt phpmyadmin
		- Cài đặt mySQL
		- Có thể sử dụng Xampp để đễ quản lý 
	* Các bước chạy:
		- Import file database.sql vào phpmyadmin 
		- Chỉ sửa các thông số trong file ".env" để phù hợp với các thông số của máy tính
		- Mở cửa sổ terminal với đường đẫn của project chạy: npm i để cài đặt các package cần thiết cho chương trình
		- Sau khi cài đặt xong, chạy lệnh: npm start để khởi động chương trình
